# Example Analysis Shape

Use this shape when producing future Serenity-style research analysis.

## Output Template

```text
Evidence base:
- priced rows:
- benchmark:
- minimum sample threshold:

Validated patterns:
- pattern:
- n:
- median 20d excess return:
- excess win rate:
- median drawdown:
- representative source URLs:
- manual review status:

Weak hypotheses:
- hypothesis:
- why weak:
- what would need verification:

Failure modes:
- broad themes that failed:
- context tickers mistaken as signals:
- duplicate/thread inflation:
- proxy/liquidity caveats:

Reusable method:
- demand shock:
- bottleneck layer:
- scarce capability:
- underpriced beneficiary type:
- evidence stack:
- signal classification:
- benchmark audit:
```

## Tone

Write as research process, not promotion. Avoid buy/sell language. Prefer: "This event cluster suggests..." and "This requires manual review...".
